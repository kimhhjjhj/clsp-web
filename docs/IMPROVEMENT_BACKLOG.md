## Tier 2 — 데이터 활용 (도메인 임팩트)

- [ ] **전사 통합 대시보드 `/dashboard`** — 모든 프로젝트 합산 KPI, 공종별 누적 인일, 월별 트렌드
  - 완료 조건: 프로젝트 3+개 상태에서 한 화면에 요약 표시
- [ ] **프로젝트 비교 뷰** — 2~3개 프로젝트 나란히 (공기/생산성/공종 분포)
  - 완료 조건: SITE1 vs SITE2 vs 상봉 비교 화면 동작
- [ ] **관리자 일괄 승인 UI** — 현재 pending 제안들을 검증 조건(proj≥2, samples≥30)으로 일괄 승인 버튼
  - 완료 조건: `/admin/productivity` 에 "검증된 것 일괄 승인" 버튼 + 서버 배치 API
- [ ] **실적 진도율 자동 계산** — WBS × 일보 × 물량 매칭으로 "기초 72% 완료" 같은 %
  - 완료 조건: 3단계에 공종별 진도율 + 예상 완료일 표시
- [ ] **공기 지연 예측** — 현재 진도 vs 계획 vs 회사 표준 → 조기 경보
  - 완료 조건: 프로젝트 상세 상단에 "현재 속도 유지 시 N일 지연" 뱃지
- [ ] **주간/월간 자동 리포트** — 특정 기간 분석 결과를 PDF/엑셀 1장으로 추출
  - 완료 조건: "이번 달 리포트 생성" 버튼 → PDF 다운로드
- [ ] **CPM → MS Project(.mpx) 내보내기** — 현장에서 요구되는 표준 포맷
  - 완료 조건: 1단계 PDF 다운로드 옆에 "MSP 내보내기" 버튼

## Tier 3 — 프로세스맵핑 전문화 (강의자료 준수 심화)

- [ ] **Pull 역산 자동 배치** — 마일스톤을 먼저 고정 → duration으로 역산해 startDay 자동 계산
  - 완료 조건: "역산 배치" 버튼 누르면 마일스톤 기준 모든 선행 카드 startDay 재계산
- [ ] **조닝(Zone) 공식 지원** — 프로젝트 조닝을 DB에 저장, 레인을 공종×구역으로 자동 분할
  - 완료 조건: 프로젝트 설정에서 구역 정의 → Pull Planning에 자동 반영
- [ ] **Handoff 링크 마커** — 링크 편집에서 "이 지점은 인수인계" 토글, 강조 시각화
  - 완료 조건: 체크 시 주황 두꺼운 선 + 별표 아이콘
- [ ] **PDF 회의자료 출력** — Pull Planning 보드 1페이지 A3 landscape PDF (회의체 배포용)
  - 완료 조건: 사이드 툴바 "회의자료 PDF" → 인쇄 가능 레이아웃
- [ ] **실시간 협업 (WebSocket/Yjs)** — 여러 사용자 동시 편집, 선택/커서 표시
  - 완료 조건: 두 브라우저에서 동시 편집 시 실시간 반영
- [ ] **템플릿 라이브러리** — 공동주택/스튜디오/오피스용 표준 프로세스맵 저장·불러오기
  - 완료 조건: "템플릿으로 저장" / "템플릿에서 시작" 2개 기능 동작
- [ ] **도면 위 레이어 배치** — 프로젝트 DXF 평면 위에 카드·그룹을 공간적으로 배치
  - 완료 조건: 4번째 뷰 "도면 뷰" 탭 + DXF 배경 + 카드 오버레이

## 작업 이력 (완료된 굵직한 것들)

참고만 — 루프는 이 섹션을 건드리지 않는다.

- ✅ 1단계 CPM + WBS + 몬테카를로 + 생산성 + 자원계획 + 회사실적
- ✅ 2단계 R&O + 단축공법 + 시나리오 비교 + 베이스라인 비교
- ✅ 2단계 프로세스맵 (타임라인 + 플로우 + Pull Planning 3뷰)
- ✅ 3단계 일보 + 엑셀 임포트(파주/상봉 포맷) + 사진 업로드·갤러리
- ✅ 4단계 분석 (공종/월/요일/날씨/자재/장비 + 위치·부위·작업종류 자동 추출)
- ✅ 관리자 생산성 제안·승인 플로우
- ✅ 공종 별칭 정규화 (B안)
- ✅ DXF 도면 → 면적·둘레 자동 추출
- ✅ 프로세스맵 카드 상세 양식 (강의자료 준수 — 작업사항/작업구간/요청사항)
